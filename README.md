# HOS05 – Python Functions Practice

## Overview
This project includes five short Python exercises that practice writing and calling functions.  
Each function demonstrates basic concepts like parameters, default values, loops, and conditionals.

## Exercises
1. **make_shirt()** – Prints a shirt size and custom message using positional and keyword arguments.  
2. **make_shirt_default()** – Uses default values for size and message.  
3. **describe_city()** – Prints the city name and country (Vietnam as default).  
4. **show_messages()** – Loops through a list and prints each message.  
5. **count()** – Counts uppercase and lowercase letters in a string.
